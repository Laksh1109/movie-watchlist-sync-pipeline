# 🎬 Movie Watchlist Sync Pipeline

## 📌 Overview
This project is a Python-based data pipeline that enriches a movie watchlist using an external API and generates ranked recommendations.

## ⚙️ Features
- CSV data ingestion
- API integration (OMDb)
- Data cleaning & enrichment
- SQLite database storage
- Movie ranking system
- Weekend playlist generation
- CSV export

## 🛠 Tech Stack
- Python
- Pandas
- SQLite
- OMDb API

```bash
python main.py
